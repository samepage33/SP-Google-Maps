SP Google Maps is a lightweight plugin for creating google maps with street view for your WordPress Site.

== Description ==
<p><strong>SP Google Maps</strong> is a lightweight plugin for creating google maps with street view for your WordPress Site. You can create unlimited map with this plugin. and the user can also calculate the route to your place from your website based on <code>Google Maps</code> and <code>HTML5's Geo-Location API</code></p>
<strong>This is a easy to use plugin includes:</strong>
<ul>
<li>Create <u>Unlimited</u> Maps</li>
<li>Native ShortCode Support</li>
<li>Google Maps Preview in New Maps Page For Pick The Marker Location From The Map</li>
<li>Also Have Google Street View With <code>Point of View</code> Calculation Function.</li>
<li>Google Maps Style API Supported.</li>
<li>Route Calculator On Map Page</li>
<li>Translation Ready</li>
</ul>
<strong>Limitation &amp; Upcoming Features:</strong>
<ul>
<li>Uses Google Maps JavaScript (Experimental) API V3</li>
<li>This Version Doesn't Have Support For API Key. Which is not Required For Google Maps JavaScript API V3 But Recommended (<a href="https://developers.google.com/maps/signup">Source</a>). This Feature Will Be Available In Next Update.</li>
<li>Can't Use Multiple Maps in Same Page or Post. This will be fixed in next update.</li>
<li>No support for custom css. This will be available in next update</li>
<li>For Infowindow use the post content editor in text mode; you can use html but some charecter ":" and new line charecter Generated by wordpress may breake the javascript. Prepare your info window markup as a single line html. We are trying to fix this bug as soon as possible.</li>
</ul>

== Installation ==
**This section describes how to install the plugin and get it working:**
`1. Download & unzip to WordPress plugin directory.
2. Login to WordPress admin panel and active the plugin.
3. A new menu item [SP Google Maps] will appear under [Pages].`

**Upgrade:**
`1. Deactivate WP-LinkCheck.   
2. Download the latest WP-LinkCheck , unzip and over write the old files.       
3. Activate WP-LinkCheck.
--OR--
Simple using Wordpress automatic upgrade.`
